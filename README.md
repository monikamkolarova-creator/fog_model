# Fog Model MATLAB Repository
This repository contains MATLAB live scripts for simulating the particle size distribution of advection type of fog. The input parameters are fog height and number of curves per height.

## Files

### Curve_Generator.mlx 
#### Matlab file for 3D_curve_reconstruction

### cross_section_calculation

## Requierements

- MATLAB 2022a or later

## Usage
1. Download the folder corresponding to a task you want to run.
2. Open the appropriate '.mlx' in MATLAB
3. Make sure your path in MATLAB is set up to this folder so data files can be accessed
4. Run the script sequentially
